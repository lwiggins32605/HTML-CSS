WeatherPy Analysis


Latitude vs. X

The purpose of this project was to do a weather analysis on cities around the world based on their distance from the equator. As a class we were provided with citipy, which contained code that would help us get a list of cities to use for the analysis. A starter code that would create a framework to build on, and some example images to see what the goal was.

The first thing I did was generate a random list of cities using citipy, whiche yielded around 500+ cities to work with. After the list was created I built a python code that was able to parse through the Json format from the API (OpenWeatherMap API).

The code was designed to make succesive API calls and push all the data into Pandas data frames. Once all the requests were made I cleaned up the data using pandas library and began the analysis.

I looked at four different weather elements (WindSpeed, Cloudiness, Humidity and Temperature), and compared them to the cities latitude in order to see if there was any correlation between eachother.

Finally I used matplotlib to plot the data into useful visualisations to understand the data better.
